usage: generate_email.py [-h] [-f FILE] [-n FIRST_NAME LAST_NAME] -d DOMAIN

Generate email addresses from names.

options:
  -h, --help            show this help message and exit
  -f FILE, --file FILE  Path to the text file containing names (first last).
  -n FIRST_NAME LAST_NAME, --name FIRST_NAME LAST_NAME
                        First and last name to generate email addresses.
  -d DOMAIN, --domain DOMAIN
                        Email domain (e.g., domain.com).
